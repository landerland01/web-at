Обо мне

Мне 57 лет, в молодости я получил высшее техническое образование, закончил физико-технический факультет Новосибирского Электротехнического Института. Где изучал основы программирования и алгоритмы. По работе я не был связан с программированием. Но у меня был опыт программирования на языках Бейсик, Си, и ассемблере для микропроцессоров.


За время обучения по программе "Верстка CSS и программирование на языке JavaScript" я освоил основные навыки по верстке веб-сайтов с использованием иструментов Cascading Style Sheets (CSS) и «языка гипертекстовой разметки» HyperText Markup Language (HTML) и языка JavaScript а так же Figma — графический редактор для совместного проектирования сайтов, приложений и других дизайнерских продуктов. Изучил систему контроля версий на основе GitHab. За время обучения мною были созданы следующие проекты:

Первым моим проектом была верстка сайта. Необходимо было создать сайт, используя дизайн Figma в качестве макета. При его написании я использовал html, CSS и графический редактор Figma. Ссылка на депозиторий в GitHab: https://github.com/landerland01/website

Еще одним проектом было создание веб калькулятора. При его написании я использовал html, CSS и JavaScript. Ссылка на депозиторий в GitHab: https://github.com/landerland01/calculator
